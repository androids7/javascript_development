android.js库
android.js库是一个可以调用安卓api的库
它使用的是javascript脚本的全局变量
因此在命名函数的时候必须要避开
否则后面定义的相同函数名会覆盖掉原本的函数
#----------#
不要使用app和window.app变量
这是安卓API对象
#---------#
>>>>>文件读写接口<<<<<<
//读取一个文件内容
var string=ReadFile("/sdcard/xxx.txt");
//写入一个文件内容 WriteFile("/sdcard/1.txt","我是文字");
//获取文件列表
var obj=FileList("/sdcard");
//这里返回的是一个json对象
/*
访问它需要获取它的长度
这个对象有dir和file属性
使用全局变量函数getJsonObjLength(Json)返回它的长度
len=getJsonObjLength(obj);
*/
/*
循环从这个对象取出文件的列表
for(i=0;i { document.write(obj.file[i]+" }
这样会输出这个对象的file属性
当然输出文件夹的列表也是这样，不过使用的是dir属性
*/
>>>>>显示Toast<<<<<
Toast("我是文字",0);
//第二个参数只能是0和1
/*
0参数表示短时显示
1参数表示长时显示
*/
